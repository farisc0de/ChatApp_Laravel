# ChatApp_Laravel

ChatApp rewrite in Laravel Reverb with AES Encryption

## Features

1. Multiple Chat Rooms
2. AES Encryption
3. User Authentication
4. User Registration
5. User Profile
6. User Online/Offline Status
7. User Typing Status
8. Emoji Support
